# 🎲 DevSorteio - Random Number Generator

A clean and efficient web tool developed to generate random numbers within a user-defined range. This project focuses on DOM manipulation and provides a seamless user experience by displaying results directly on the interface.

## ✨ Features

* **Custom Range:** Allows users to define both minimum and maximum values for the draw.
* **Instant Result:** Displays the generated number directly on the screen, replacing traditional browser alerts for a better UI.
* **Validation Logic:** Built-in check to ensure the minimum value is always lower than the maximum, preventing calculation errors.
* **Interactive Design:** Hover and active states on buttons to provide clear visual feedback to the user.
* **Fully Responsive:** Optimized layout that works perfectly on mobile phones, tablets, and desktops.

---

## 🛠️ Technologies Used

This project was built to practice clean code and direct DOM interaction:

* **HTML5:** Semantic structure for a clear and accessible interface.
* **CSS3:** Custom styling using **Flexbox** for alignment, including glassmorphism-inspired inputs and shadow effects.
* **JavaScript (ES6+):** * Dynamic DOM manipulation to update the result field in real-time.
    * Mathematical logic using `Math.random()` and `Math.floor()` for precise randomization.
    * Input handling and value validation to ensure a bug-free experience.
